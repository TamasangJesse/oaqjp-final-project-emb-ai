# oaqjp-final-project-emb-ai

## Emotion Detection Web Application

An AI-based web application that performs emotion detection on customer feedback using the Watson NLP Embedded AI Library.

## Project Name
**Emotion Detector**

## Description
This application analyzes text input and detects emotions such as anger, disgust, fear, joy, and sadness using the Watson NLP library, deployed as a Flask web application.

## Tasks Completed
1. Clone the project repository
2. Create an emotion detection application using Watson NLP library
3. Format the output of the application
4. Package the application
5. Run Unit tests on the application
6. Web deployment of the application using Flask
7. Incorporate error handling
8. Run static code analysis
